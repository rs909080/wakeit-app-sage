SECTION 0 â€” PROJECT OVERVIEW

0.1  App Summary

Wakeit is a group alarm sync app. One person (the admin) sets an alarm with a custom tone and it rings on every group member's device simultaneously. The alarm cannot be dismissed by tapping a button â€” each member must send a real SMS to the Wakeit system number to confirm they are awake. The admin gets a live dashboard showing who has woken up and who is still sleeping.

0.2  Core Features

Group-based alarm sync via 6-digit invite code

Custom alarm tones: upload audio file or record voice message

Sync online, ring offline (alarm data and tone cached locally after sync)

Auto day/night theme switching based on device time (6AMâ€“6PM day, else night)

SMS-confirm dismiss: alarm only stops when the member sends an SMS to the Wakeit Twilio number

Admin wake dashboard: real-time view of who is awake vs still sleeping, with Awake/Sleeping filter tabs

Admin nudge: admin can send a follow-up push buzz to any sleeping member â€” that is the only admin action

0.6  How SMS Dismiss Works (Architecture)

This is the most important feature. Understand the full flow before building:

Admin creates a group. A dedicated Twilio virtual phone number is assigned to the group and stored in the groups table (or one shared Twilio number is used for all groups, with the group identified by the sender's registered phone number).

Admin sets an alarm. All group members receive it via Supabase Realtime. Each member's device schedules the alarm locally.

At alarm time, every member's alarm screen rings. The screen shows: the Twilio number to SMS, the required message text (e.g. 'WAKE'), and a live countdown.

Member opens their native SMS app (link opens it pre-filled), sends the SMS.

Twilio receives the SMS and POSTs to the Supabase Edge Function webhook URL.

The Edge Function looks up the sender's phone number in the profiles table, identifies which alarm is active for their group, and updates the alarm_wake_status table: status = 'awake' for that user.

Supabase Realtime fires a change event. All group members (including the sender) see the wake dashboard update in real time.

On the sender's device: the frontend is subscribed to their own wake status row. When it flips to 'awake', the alarm audio stops and the screen navigates to #/home automatically.

On the admin's screen: the wake dashboard updates live â€” the member moves from the Sleeping list to the Awake list. Admin cannot dismiss anyone else's alarm â€” each person's alarm stops only when they personally SMS.

function authGuard(hash) {

  const publicRoutes = ['#/splash','#/onboarding','#/login','#/paywall'];

  const adminOnlyRoutes = ['#/wake-dashboard'];

  const session = supabase.auth.getSession();

  const hasPaid = localStorage.getItem('wakeit_plan');

  const isAdmin = AppState.currentGroup?.owner_id === getCurrentUserId();

  if (!publicRoutes.includes(hash)) {

    if (!session) { navigate('#/login'); return false; }

    if (!hasPaid) { navigate('#/paywall'); return false; }

  }

  if (adminOnlyRoutes.includes(hash) && !isAdmin) {

    navigate('#/home'); return false;

  }

  return true;

}

function initAlarmRinging() {

function subscribeToMyWakeStatus(alarmId) {

  const userId = getCurrentUserId();

  sb.channel('my-wake-status')

    .on('postgres_changes', {

      event: 'UPDATE',

      schema: 'public',

      table: 'alarm_wake_status',

      filter: `alarm_id=eq.${alarmId}`,

    }, payload => {

      if (payload.new.user_id === userId && payload.new.status === 'awake') {

        stopAlarmAudio();

        navigate('#/home');

        showToast('Good morning! ðŸŒ…', 'success');

      }

    }).subscribe();

}

10.3  Local Alarm Scheduling

function scheduleLocalAlarm(alarm) {

  const delay = new Date(alarm.alarm_time).getTime() - Date.now();

  if (delay < 0) return;

  const stored = JSON.parse(localStorage.getItem('wakeit_alarms') || '[]');

  stored.push(alarm);

  localStorage.setItem('wakeit_alarms', JSON.stringify(stored));

  setTimeout(() => triggerAlarm(alarm), delay);

}

// Edge function: supabase/functions/twilio-sms-webhook/index.ts

// Deploy: supabase functions deploy twilio-sms-webhook

// Set in Twilio console as the webhook URL for inbound SMS

import { serve } from 'https://deno.land/std/http/server.ts';

import { createClient } from 'https://esm.sh/@supabase/supabase-js';

serve(async (req) => {

  const form = await req.formData();

  const fromPhone = form.get('From');   // e.g. '+919876543210'

  const body = (form.get('Body') || '').trim().toUpperCase();

  if (body !== 'WAKE') {

    return new Response('<Response/>', { headers: { 'Content-Type': 'text/xml' } });

  }

  const sb = createClient(

    Deno.env.get('SUPABASE_URL')!,

    Deno.env.get('SUPABASE_SERVICE_ROLE_KEY')!

  );

  // Find the user by phone number

  const { data: profile } = await sb.from('profiles')

    .select('id').eq('phone', fromPhone).single();

  if (!profile) return new Response('<Response/>', { headers: { 'Content-Type': 'text/xml' } });

  // Find the active pending wake status for this user

  const { data: status } = await sb.from('alarm_wake_status')

    .select('id, alarm_id')

    .eq('user_id', profile.id)

    .eq('status', 'pending')

    .order('created_at', { ascending: false })

    .limit(1).single();

  if (!status) return new Response('<Response/>', { headers: { 'Content-Type': 'text/xml' } });

  // Flip status to 'awake' â€” Realtime fires immediately

  await sb.from('alarm_wake_status').update({

    status: 'awake',

    woke_at: new Date().toISOString(),

    updated_by: 'sms'

  }).eq('id', status.id);

  return new Response('<Response/>', { headers: { 'Content-Type': 'text/xml' } });

});

let mediaRecorder, recordingChunks = [];

async function startRecording() {

  const stream = await navigator.mediaDevices.getUserMedia({ audio: true });

  mediaRecorder = new MediaRecorder(stream);

  recordingChunks = [];

  mediaRecorder.ondataavailable = e => recordingChunks.push(e.data);

  mediaRecorder.onstop = async () => {

    const blob = new Blob(recordingChunks, { type: 'audio/webm' });

    await uploadTone(new File([blob], `voice_${Date.now()}.webm`));

  };

  mediaRecorder.start();

  setTimeout(() => stopRecording(), 30000);

}
